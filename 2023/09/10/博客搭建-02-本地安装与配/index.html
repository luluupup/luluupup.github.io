博客搭建-02-本地安装与配置

本文介绍在本地的安装与配置

1. 安装 Git

直接在官网下载，然后安装即可；

官网链接：https://git-scm.com/

2. 安装 node.js

和 Git 一样，直接在官网下载，然后安装；

官网链接：https://nodejs.org/en/

3. 安装 Hexo

官网链接：https://hexo.io/zh-cn/

打开官网可以看到，Hexo 的安装非常简单，只需要一行命令：

npm install hexo-cli -g

具体操作方法以 Windows 系统为例，按 Windows 键 + R ，输入 cmd，然后输入上面那一行脚本，再按回车；

完成之后，需要进行一些配置。

官网给出的之后的操作是这样的：

官网是将所有文件安装到了默认位置，当然我们也可以自己创建博客的文件夹。关于安装位置的说明以及cmd的一些操作可以参考这篇文章：博客搭建-02-1-位置选择和在cmd中打开的方法/

以下是自己创建博客文件夹的后续安装方式：

  1. 新建存储博客文件的文件夹：选择一个盘区，新建一个文件夹，更改名称为英文的名称（避免出现奇怪的 bug），例如在 D盘新创建文件夹，改名为「MyBlog」；

  2. 创建并初始化一个博客：打开这个文件夹，点击鼠标右键，选择「Git Bash here」，在弹出的命令框中输入「hexo init」；（除了「Git Bash here」之外，还有几种打开方法：博客搭建-02-1-位置选择和在cmd中打开的方法/）

  3. 安装所需组件：等待上一步成功完成之后，再继续输入「npm install」；

  4. 启动本地服务：等待上一步完成之后，继续输入「hexo server」或更简单的写法「hexo s」;

  5. 本地访问搭建的博客：打开浏览器访问「http://localhost:4000/」；

  6. 关闭本地服务：在命令行界面输入「Ctrl + c」

到此，博客已经安装并完成基础配置，能够在本地访问了。
